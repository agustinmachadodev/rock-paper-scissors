# Piedra, Papel o Tijera

Juego simple de Piedra, Papel o Tijera creado con JavaScript.

Este proyecto forma parte del curso Foundations de The Odin Project.


## Cómo funciona

- El usuario elige entre piedra, papel o tijera
- La computadora elige una opción al azar
- El juego se juega durante 5 rondas
- Al final se muestra el ganador en la consola


## Tecnologías usadas

- JavaScript
